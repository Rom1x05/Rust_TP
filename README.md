# Mini système de gestion de bibliothèque (Rust)

Projet réalisé en Rust dans le cadre d’un TP noté.

## Fonctionnalités
- Ajouter un livre
- Emprunter un livre
- Retourner un livre
- Afficher tous les livres
- Afficher uniquement les livres disponibles

## Compilation et exécution

- compilation directement dans le terminal avec :

```bash
rustc main.rs
./main
